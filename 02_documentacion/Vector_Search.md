# Vector Data and Semantic Search

## Objetivo

Implementar búsqueda semántica utilizando datos vectoriales en SQL Server 2025, como parte de las funcionalidades modernas requeridas por la rúbrica.

## Implementación

Se agregó una columna vectorial para representar características académicas de cursos. Esta representación permite comparar cursos mediante distancia vectorial y obtener resultados por similitud semántica.

## Script relacionado

- [Vector Search](../03_sql/06_json_api_vector/03_VectorSearch.sql)

## Consultas esperadas

- Búsqueda de cursos similares a IF5100.
- Ordenamiento por distancia vectorial.
- Comparación semántica entre cursos académicos.
