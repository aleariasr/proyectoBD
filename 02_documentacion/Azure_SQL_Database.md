# Azure SQL Database PaaS

## Objetivo

Desplegar la base de datos SIGAU completa, incluyendo metadatos y datos, en Azure SQL Database PaaS.

## Requisito de rúbrica

La rúbrica solicita el despliegue completo de la base de datos en Azure PaaS.

## Actividades requeridas

- Crear servidor lógico de Azure SQL.
- Crear Azure SQL Database.
- Configurar firewall.
- Migrar esquema y datos de SIGAU.
- Validar conexión.
- Validar tablas.
- Validar conteo de registros.
