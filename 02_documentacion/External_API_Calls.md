# External API Calls

## Objetivo

Implementar consumo de servicios externos desde SQL Server 2025 mediante llamadas REST, cumpliendo el requisito de External API Calls.

## Implementación

Se creó el esquema `api`, una tabla para almacenar consultas externas y un procedimiento para consumir un endpoint externo relacionado con información universitaria.

## Objetos principales

- `api.ConsultaExterna`
- `api.usp_ConsultarEndpointUniversidad`
