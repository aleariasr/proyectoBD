# Validaciones del Proyecto SIGAU

## Validaciones realizadas

- Creación exitosa de la base SIGAU.
- Creación de esquemas.
- Creación de filegroups.
- Separación de archivos físicos.
- Configuración Recovery Model FULL.
- Instalación y validación de SQL Server 2025.
- Creación de tabla In-Memory.
- Creación de vistas por tabla.
- Población mínima de 10 registros por tabla.
- Dynamic Data Masking.
- Row Level Security.
- Auditoría de servidor.
- Auditoría de base de datos.
- Backup completo.
- Restore de prueba.
- Exportación JSON.
